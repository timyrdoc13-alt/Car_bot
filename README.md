# car-channel-bot

Telegram-бот для канала объявлений: ручной пост (LLM), автопостинг с одобрением, статистика.

- Запуск: `car-bot` (в репозитории есть тестовый `.env` для быстрого деплоя; для прода смените токены и ключи, не публикуйте свои секреты).
- Mashina-отладка: `car-mashina-monitor` → http://127.0.0.1:8765/
- VPS / Docker: см. [DEPLOY.md](DEPLOY.md) (`docker compose up -d`).
